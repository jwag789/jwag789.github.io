HTML Cheat Sheet





Doctype



<!DOCTYPE html>



HTML Tag



<html></html>



Creates an HTML document



Header Tag



<head></head>



Sets title, links to CSS and other files etc.



Body Tag



<body></body>



Sets visible portion of document



Title Tag



<title></title>



Sets document title



Header tags



<h1></h1>



<h6></h6>



Header tags 1-6, 1 being the largest, 6 the smallest



Paragraph text



<p></p>



Paragraph align



<p align=?></p>



Align to left, right, or center



Bold Text



<b></b>



Italic Text



<i></i>



Line break



<br>



Font Size/Color



<font size=?></font>



<font color=?></font>



Create a link



<a href="URL"></a>



Mail link



<a href="mailto:EMAIL></a>



Add image



<img src="name">
